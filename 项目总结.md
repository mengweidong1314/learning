# 📄 PDF下载功能详细代码 - 退款申请系统

## 🎯 项目概述

本项目是一个完整的PDF下载功能演示系统，以退款申请为业务场景，使用Spring Boot + iText 7技术栈，实现了专业的PDF生成和下载功能。

## 🏗️ 项目架构

```
PDF下载系统
├── 表现层 (Controller)
│   └── RefundController - REST API接口
├── 业务层 (Service)
│   ├── RefundApplicationService - 业务逻辑
│   └── PdfService - PDF生成服务
├── 数据层 (Repository)
│   └── RefundApplicationRepository - 数据访问
└── 实体层 (Entity)
    └── RefundApplication - 退款申请实体
```

## 🛠️ 核心技术栈

- **Spring Boot 3.2.1** - 主框架
- **iText 7.2.5** - PDF生成库
- **Spring Data JPA** - 数据持久化
- **H2 Database** - 内存数据库
- **Lombok** - 代码简化
- **Maven** - 项目管理

## 🔧 核心代码结构

### 1. 实体类设计 (RefundApplication.java)
```java
@Entity
@Table(name = "refund_application")
public class RefundApplication {
    private Long id;                    // 主键ID
    private String applicationNo;       // 申请单号
    private String applicantName;       // 申请人姓名
    private String applicantPhone;      // 申请人手机号
    private String originalOrderNo;     // 原订单号
    private BigDecimal refundAmount;    // 退款金额
    private String refundReason;        // 退款原因
    private String bankCardNo;          // 银行卡号
    private String bankName;            // 开户行
    private Integer status;             // 状态 (0:待审核 1:已通过 2:已拒绝)
    private LocalDateTime applicationTime; // 申请时间
    private LocalDateTime processTime;     // 处理时间
    private String remarks;             // 备注
}
```

### 2. PDF服务接口 (PdfService.java)
```java
public interface PdfService {
    // 生成单个退款申请PDF
    byte[] generateRefundApplicationPdf(RefundApplication refundApplication);
    
    // 生成退款申请列表PDF
    byte[] generateRefundApplicationListPdf(List<RefundApplication> refundApplications);
    
    // 生成统计报告PDF
    byte[] generateRefundStatisticsPdf(List<Object[]> statusCounts);
    
    // 生成自定义内容PDF
    byte[] generateCustomContentPdf(String title, String content);
}
```

### 3. 控制器设计 (RefundController.java)
```java
@RestController
@RequestMapping("/api/refunds")
public class RefundController {
    
    // 下载单个退款申请PDF
    @GetMapping("/{id}/pdf")
    public ResponseEntity<byte[]> downloadRefundApplicationPdf(@PathVariable Long id);
    
    // 下载退款申请列表PDF
    @GetMapping("/list/pdf")
    public ResponseEntity<byte[]> downloadRefundApplicationListPdf(
        @RequestParam(required = false) Integer status,
        @RequestParam(required = false) String startTime,
        @RequestParam(required = false) String endTime
    );
    
    // 下载统计报告PDF
    @GetMapping("/statistics/pdf")
    public ResponseEntity<byte[]> downloadRefundStatisticsPdf();
    
    // 生成自定义内容PDF
    @PostMapping("/custom/pdf")
    public ResponseEntity<byte[]> generateCustomPdf(@RequestBody Map<String, String> request);
}
```

## 📊 API接口详细说明

### 1. 获取数据接口
| 方法 | 路径 | 描述 | 参数 |
|------|------|------|------|
| GET | `/api/refunds` | 获取所有退款申请 | 无 |
| GET | `/api/refunds/{id}` | 获取单个退款申请 | id: 申请ID |
| GET | `/api/refunds/statistics` | 获取统计数据 | 无 |

### 2. PDF下载接口
| 方法 | 路径 | 描述 | 参数 |
|------|------|------|------|
| GET | `/api/refunds/{id}/pdf` | 下载单个申请PDF | id: 申请ID |
| GET | `/api/refunds/list/pdf` | 下载列表PDF | status, startTime, endTime |
| GET | `/api/refunds/statistics/pdf` | 下载统计报告PDF | 无 |
| POST | `/api/refunds/custom/pdf` | 生成自定义PDF | title, content |

### 3. 查询参数说明
- **status**: 状态筛选 (0:待审核, 1:已通过, 2:已拒绝)
- **startTime**: 开始时间 (格式: yyyy-MM-dd HH:mm:ss)
- **endTime**: 结束时间 (格式: yyyy-MM-dd HH:mm:ss)
- **applicantName**: 申请人姓名
